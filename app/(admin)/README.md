# Back-office NOVANOX

Ce groupe contient les pages sous `/admin` réservées aux membres invités de l’équipe NOVANOX.

Les droits dépendent des rôles `SUPER_ADMIN`, `ADMIN` et `EMPLOYEE`. Les opérations privilégiées doivent passer par du code serveur fiable et être journalisées lorsqu’elles touchent aux clients, projets, devis, factures, fichiers ou comptes.
